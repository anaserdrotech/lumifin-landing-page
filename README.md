# LumiFin - Landing Page Fintech

Uma landing page moderna e responsiva para a plataforma **LumiFin**, uma solução de fintech voltada para controle financeiro pessoal.

## 📋 Descrição do Projeto

LumiFin é uma plataforma que permite aos usuários gerenciar suas receitas, despesas, investimentos e metas de forma integrada. Esta landing page apresenta os principais diferenciais e funcionalidades da plataforma.

## 🎯 Funcionalidades

- **Múltiplas Contas**: Gerencie contas corrente, poupança e carteira em um único lugar
- **Transações Categorizadas**: Organize suas movimentações financeiras por categorias customizáveis
- **Metas Financeiras**: Planeje seus objetivos de curto, médio e longo prazo
- **Dashboard Intuitivo**: Visualize seu saldo total, entradas, saídas e progresso de metas

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e bem organizada
- **CSS3**: Estilos customizados e responsivos
- **Tailwind CSS**: Framework CSS para construção do layout
- **Google Fonts**: Tipografia moderna (Poppins e Inter)

## 📱 Responsividade

A landing page é totalmente responsiva e funciona perfeitamente em:
- 📱 Dispositivos móveis (320px+)
- 📱 Tablets (768px+)
- 💻 Desktops (1024px+)

## 🚀 Como Usar

1. **Clone ou baixe o repositório**
   ```bash
   git clone https://github.com/seu-usuario/lumifin-landing-page.git
   cd lumifin-landing-page
   ```

2. **Abra o arquivo HTML**
   - Simplesmente abra o arquivo `index.html` em seu navegador preferido
   - Não há necessidade de servidor local ou build process

3. **Visualize em diferentes tamanhos de tela**
   - Use as ferramentas de desenvolvedor do navegador (F12)
   - Teste a responsividade em diferentes resoluções
